# Introduction

## CKA 시험
- 온라인 시험
- 비용 300달러
- 2번의 기회
- 객관식 아님. 
- 시험시간 : 3시간
- 공식 문서 페이지를 활용할 수 있음. 이거 사용하는 법 알려줄거임
- 시험 신청시 : DEVOPS15 코드 사용해서 15% 할인 받기
